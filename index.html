Follow your Favorite Devs!

Feature Highlights

Save GitHub Profiles

Do you have rockstar developers you like to follow on Github? This app utilizes the Github API so you can save an easy to keep a pulse on the coding activity of your favorite developers.

Make Notes, Edit & Delete

This app makes it easy for you to make notes on any of your favorite developers work! Editing and deleting notes has never been so easy! Simply edit any notes that were made or delete them on the spot if necessary!

Follow Commit History

Be able to keep a close eye on your favorite developers commit history! Youll be able to see all commits made to GitHub and have an organized view of all the commits made.

View Most Popular User Repos

Youll be able to have a view of what the most popular popular user-owned repositories! Keep tabs on popular repositories that you have contributed to but not authored.

Stay Up To Date

Have an organized view of what your favorite developers are up to, so you can stay up to date with all the work thats being done in the programming world.

Best Practices

Great effort was put in place to comply with accessibility and symantic best practices. Our single page web UI scores 100 on accessibility.
